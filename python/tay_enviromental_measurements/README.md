

## Data

The dataset used for this project was the 'River Tay, Scotland, water chemistry and greenhouse gas measurements (2009-2010)'. This resource is available under [the Open Government Licence v3 (OGL)](https://www.nationalarchives.gov.uk/doc/open-government-licence/version/3/) and contains data supplied by UK Centre for Ecology & Hydrology.

Pickard, A.E.; Skiba, U.M.; Carvalho, L.; Heal, K.V.; Rees, R.M.; Harley, J.F. (2019). River Tay, Scotland, water chemistry and greenhouse gas measurements 2009-2010 NERC Environmental Information Data Centre. https://doi.org/10.5285/a61da7da-b7ef-40b7-a324-c3711ef81207 (Accessed: 20 March 2026)
