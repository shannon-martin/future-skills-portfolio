
# Health Status Dashboard
My first dashboard shows the health status distribution by age groups using a 100% stacked bar chart, and the percentage of people that reported each health status using a donut chart. The dashboard also includes a slicer to dynamically view selected age group data and card display to show the current selected number of people in the selected category. I also included some notes of the insights taken from the visualisations.
