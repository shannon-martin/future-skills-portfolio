# Heath & Demographics Dashboard
This dashboard contains a title, 3 filters (Year, Region, and Gender), a dynamic card showing the total count of health status responses, a stacked bar chart showing the distribution of health status by age group, and a donut chart showing the percentage distribution of currently selected health status.

<img width="1311" height="736" alt="image" src="https://github.com/user-attachments/assets/80f7be72-0abc-490c-9e13-d13c04d57fe5" />
I customised a theme to display consistent colours across all my visualisations. I decided that a colour scale of: Dark Green > Green > Yellow > Orange > Red was the clearest way to colour the different health statuses. I also created a custom sort order for the legend displays.


# Health Status Dashboard
This dashboard shows the health status distribution by age groups using a 100% stacked bar chart, and the percentage of people that reported each health status using a donut chart. The dashboard also includes a slicer to dynamically view selected age group data and card display to show the current selected number of people in the selected category. I also included some notes of the insights taken from the visualisations.
