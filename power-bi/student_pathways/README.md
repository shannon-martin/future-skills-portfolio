**Note:** The SFC advises that earlier Scottish Wider Access Programmes (SWAP) enrolments were underreported, so part of the observed increase in the Access to HE pathway reflects improved data alignment rather than purely underlying growth.

To create this visual, the individual categories of learners were grouped into pathways to represent the broad groups that describe the learner’s status at the point of enrolment to help understand the patterns in higher education vs HND pathways in Scotland. This approach enabled a consistent comparison of pathways over time to examine trend. Below is a table containing the mapping used:


| Category of student | Pathway |
|---------------------|---------|
| Full-time college based not included above | College-based |
| Migrant worker | Other/Unspecified |
| Not Known | Other/Unspecified |
| Not registered unemployed but not working | Unemployed |
| PI-P7, S1–S6, but not on a school link programme | School-age |
| Permanent or temporary employment | Employed |
| Primary school pupil | School-based |
| Registered unemployed - not receiving jobseekers allowance | Unemployed |
| Registered unemployed - receiving jobseekers allowance | Unemployed |
| Retired | Other/Unspecified |
| School based S1 | School-based |
| School based S2 | School-based |
| School based S3 | School-based |
| School based S4 | School-based |
| School based S5 | School-based |
| School based S6 | School-based |
| Student on European exchange (e.g. ERASMUS, COMETT) | Other/Unspecified |
| Student on Scottish wider access programme (SWAP)* | Access to HE |
| Student on Youth Training / Employability Fund / Modern Apprenticeship | Apprenticeship/Training |
| Student on other access course for Higher Education (all modes) - other than conventional SCE/GCE/GCSE studies | Access to HE |
| Student on other government training scheme | Apprenticeship/Training |
| Winter leaver | Other/Unspecified |


Table 13 from the ‘College-Statistics-Background-Tables.xlsx’ file was used to create this visualisation. The full Background Tables dataset provided by the Scottish Funding Council is available at: https://www.sfc.ac.uk/publications/college-statistics-2023-24/
