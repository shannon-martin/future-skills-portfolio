# future-skills-portfolio
A collection of mini‑projects, visualisations, and other submissions completed during my Future Skills Programme as learning evidence towards a Data Science accreditation. Includes Python notebooks, Power BI dashboards, Excel analyses, GIS work, automation, and supporting documentation. Larger projects are hosted in separate repositories.

# Purpose of This Repository
This repo is designed to function as a **learning portfolio** and a record of my progress throughout the programme. It also serves as a demonstration of my ability to use GitHub for version control, documentation, and professional project organisation. While many of these tasks are small or exploratory, together they showcase core skills such as:
- Data cleaning and preparation
- Exploratory data analysis
- Data visualisation and dashboard development
- Python scripting and automation
- Working with geospatial data
- Communicating insights clearly and effectively
