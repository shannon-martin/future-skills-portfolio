# Council Tax by Geography: Scotland's Housing Landscape
This project aims to analyse how the council tax system that operates in Scotland using valuations from 1991 relates to the current housing market, average incomes, and geographical disparities. Using ArcGIS, the project maps council tax bands by comparing them with current prices for houses and average incomes within different councils.  
This project reveals an important problem that needs to be solved - council taxes often do not match the median price of a house in certain local authorities, nor the cost level. There can be situations where poorer authroities have higher taxes compared to their median incomes, while richer authroities do not always have corresponding taxes.  
By visualising these patterns, this project aims to improve understanding of how geography influences affordability and aims to support discussion around whether council tax should better reflect modern economic conditions.
